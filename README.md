# Portfólio — Vitor

Este repositório contém um template de portfólio pessoal simples e responsivo usando HTML, CSS e JavaScript puros. Personalize e publique facilmente com GitHub Pages.

## Estrutura

- `Index.html` — Página principal do portfólio
- `styles.css` — Estilos e layout responsivos
- `script.js` — Interações (menu, tema, smooth scroll)

## Personalização rápida

1. Abra `Index.html` e altere o `brand` e o nome exibido no hero.
2. Edite as seções `Sobre`, `Projetos`, `Habilidades` e `Contato` conforme necessário.
3. Troque os links (Email, GitHub, LinkedIn) para os seus.
4. Substitua miniaturas e imagens nos cards por imagens reais.

## Deploy (GitHub Pages)

1. Crie um repositório no GitHub e envie os arquivos.
2. Vá para `Settings` do repo → `Pages` → Selecione `main` branch (ou `gh-pages`) e clique em `Save`.
3. Aguarde alguns minutos e acesse `https://<seu-usuario>.github.io/<seu-repo>/`.

## Como testar localmente

Você pode abrir `Index.html` diretamente no navegador ou iniciar um servidor local simples com Python:

```powershell
python -m http.server 8000
```

Em seguida, abra `http://localhost:8000` no navegador.

## Dicas

- Para formulários reais, integre um backend ou use serviços de formulário como Formspree.
- Considere adicionar imagens e uma versão do `resume` em PDF para download.
- Se desejar, adicione suporte a i18n para o conteúdo em Português/Inglês.
